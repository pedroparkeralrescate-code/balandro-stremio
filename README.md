# Balandro Stremio Addon

Addon de Stremio con todos los proveedores de Balandro (127 channels + 92 servers).

## Instalación en Stremio

1. Despliega este proyecto en Vercel
2. En Stremio, ve a **Add-ons > Install from URL**
3. Pega: `https://YOUR-PROJECT.vercel.app/manifest.json`
4. ¡Disfruta de todo el contenido en español!

## Desarrollo Local

```bash
# Instalar dependencias
pip install -r requirements.txt

# Ejecutar localmente con Vercel CLI
npm i -g vercel
vercel dev
```

## Providers Incluidos

- **127 channels**: Cuevana3, PelisPlus, Gnula, HDFull, CineCalidad, y muchos más
- **92 servers**: Fembed, StreamTape, Doodstream, Mega, Mixdrop, y más

## Características

- ✅ Películas y series en español
- ✅ Múltiples idiomas (Latino, Español, VOSE)
- ✅ Múltiples calidades (CAM, HD, FullHD, 1080p)
- ✅ Búsqueda automática en todos los proveedores
- ✅ Resolución automática de enlaces
